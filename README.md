# nifi-etl-docker-setup

Apache NiFi Docker Setup

This project sets up Apache NiFi for ETL processing using Docker.
It ensures all team members use the same version without dependency conflicts.

Prerequisites

Docker installed

Docker Compose installed

Step 1: Clone the Repository

Step 2: Start Apache ""NiFi docker-compose up -d""

Step 3: Access NiFi ""http://localhost:8088/nifi""

Step 4: Stop NiFi ""docker-compose down""

Version Used

Apache NiFi Version: 1.23.2

Notes

All data is stored inside the nifi_data folder.

No manual installation of Java or NiFi is required.

Works on Linux, Mac OS, and Windows.
